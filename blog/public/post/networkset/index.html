Openwrt 如何下发不同的DNS和网关#

修改/etc/config/network 文件 config ‘interface’ ’lan’
option ’type’ ‘bridge’
option ‘ifname’ ’eth0.0’
option ‘proto’ ‘static’
option ’netmask’ ‘255.255.255.0’ option ‘dns’ ‘208.67.222.222’
option ‘gateway’ ‘192.168.3.1’
option ‘ipaddr’ ‘192.168.3.250’ ————————————————
